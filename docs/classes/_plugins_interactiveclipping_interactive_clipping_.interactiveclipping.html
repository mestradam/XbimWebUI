clip

  • clip(): void
    • Defined in plugins/InteractiveClipping/interactive-clipping.ts:42

    Use this method to start interactive clipping of the model. This is based on SVG overlay so SVG support is necessary for it. But as WebGL is more advanced technology than SVG it is sound assumption that it is present in the browser. Use Viewer.check() to make sure it is supported at the very beginning of using of Viewer. Use {@link Viewer#unclip unclip()} method to unset clipping plane.
